## Game Energy App <br> 
Gjord av Ludwig och yusuf

# Produktbeskrivning

En applikation för ett mätta användning av energi i form av kwh med hjälp av datorns användning %

# Produktmål

Vårt mål är 150 användare inom 3 veckor med 90% gladda med servicen


# Installation/Execution Guide
## Collector Commands to run
``` cd Project/Collector```
``` python data.py```
## Backend Commands to run
``` cd Project```
``` python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000```

## Frontend Commands to run

``` cd Project/frontend ```
``` npm run dev ```
